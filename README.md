# Portfolio

Personal portfolio website for **Shreyash Pattewar** — Mobile & AI Developer.

Built with vanilla HTML, CSS, and JavaScript. No frameworks, no build step.

## 🔗 Live

https://shreyashp47.github.io

## 🚀 Quick start

```bash
open index.html
```

## 📁 Structure

- `index.html` — page structure, inline theme script
- `static/css/` — base, components, responsive
- `static/js/` — config, themes, effects, render, navigation, main
- `static/assets/` — resume, profile image
- `SS/` — screenshots
